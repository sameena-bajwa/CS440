Problem Definition

For our first programming assignment, we were instructed to design and implement a simple neural network and train it on different datasets.

Method and Implementation

The main part of the coding for this assignment was creating three methods for the logistic regression and neural network classifier. These methods were initially built for the logistic regression classifier and then were appropriately modified for the neural network.

§  Compute_cost(self, X, y) – Returns average cost per data sample

o   Logistic regression classifier

§  Uses forward propagation to compute the softmax scores for X

o   Neural network classifier

§  Added an activation function to link with the new hidden layer

§  Predict(self, X) – Returns an array of prediction labels for the input

o   Logistic regression classifier

§  Uses forward propagation to run our input through a softmax function

o    Neural network classifier

§  Activation function is performed on the new hidden layer

§  The output of the activation function is run through a softmax function

§  Fit(self, X, y) – Learns model parameters to fit data

o   Logistic regression classifier

§  For a specified number of iterations:

·      Uses forward propagation to compute the softmax scores for X

·      Converts y into its corresponding vector representation

·      Finds the difference between the actual value obtained and the predicted value - beta

·      Uses backwards propagation to compute the cost gradient for weight and bias

·      Updates weight and bias accordingly

o   Neural network classifier

§  For a specified number of iterations

·      Two weights and biases to update

·      Cost gradient for weight and bias of hidden layer computed according to the result of the activation function

·      Second beta value was computed using the derivative of the activation function

·      Cost gradient for weight and bias of output layer computed according to the second beta value

·       Updates weights and biases accordingly
